# AI Study Assistant 📚

A local GenAI application powered by Llama3 and Ollama.

## Features
- Summarize study material
- Generate study questions
- Beginner-friendly explanations
- PDF upload support
- Local AI inference using Ollama

## Technologies
- Python
- Streamlit
- Ollama
- Llama3
- PyPDF

## Installation

```bash
git clone <repo-url>
cd ai-study-assistant
pip install -r requirements.txt
ollama run llama3
python3 -m streamlit run app.py
```
